## Title formula (max 80 chars)

[Card/Item Name] [Set Name] [Language] [Condition/Grade] [Edition] [Year] [Key Feature]

Examples:
- Pokemon Charizard ex Surging Sparks English NM Full Art 2024
- Pokemon Mega Dream M2a Booster Box Japanese Factory Sealed 2025
- Pokemon Charizard Base Set 1st Edition PSA 9 1999

Rules:
- No ALL CAPS. No excessive punctuation. No RARE!!! or AMAZING!!!
- Pack in keywords buyers actually search -- set name in full, language, condition, year
- For sealed: include Booster Box / ETB / Booster Bundle so it shows in category searches
- For slabs: always include grader + grade (PSA 9, BGS 9.5, CGC 10)

## Item-specific rules

### Singles (NM/LP/MP)
- Always state condition in first line of description
- Never grade as NM if there is any visible wear
- Include card number and set number in item specifics

### Sealed product
- Note if outer shrink has any tears or dents (always disclose)
- Do not claim direct from Japan unless owner has confirmed for this specific SKU
- For Japanese/Asian product: default copy is sourced through our regional supplier network

### Graded slabs
- Always include cert number in title and description
- Always include verify instructions (psacard.com / beckett.com / cgccomics.com)
- Note case condition (intact / minor scratch on case -- always disclose)
- Only PSA, BGS, CGC -- never list HGA, GMG, or unverified-grader slabs
