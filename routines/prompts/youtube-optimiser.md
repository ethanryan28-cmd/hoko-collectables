# YouTube Video Optimiser -- HOKO Collectables

Use when uploading a new video or auditing existing ones.
Paste current title + context. Claude returns optimised title, description, tags, and thumbnail brief.

## Inputs

- Video type: Pack opening / Box break / Grading reveal / Market commentary / Behind-the-scenes / Buyback walkthrough / Other
- Current title (if editing existing): [paste or leave blank for new]
- What happens in the video: (set, language, notable pulls, rough outcome)
- Target audience: Australian TCG collectors / investors / general Pokemon fans / MTG / One Piece
- Call to action: Shop link / Subscribe / DM for buybacks / Notify for next drop / None

---

## Output

### 1. Optimised title (max 60 chars)

Formula: [Action] [Set Name] [Product Type] [Language if not English] [Location signal] [Year if relevant]

Examples:
- Opening Pokemon Destined Rivals Booster Box Australia 2026
- Is Surging Sparks ETB Worth It? Australian Market Check
- Selling Your Pokemon Collection? How HOKO Buybacks Work

Rules:
- Include the full set name (not abbreviations -- viewers search the full name)
- Include Australia or Australian in every AU market video
- No ALL CAPS, no clickbait, no excessive punctuation
- Do not spoil a pull in the title unless it was genuinely exceptional

### 2. Description

First 150 chars are shown before Show more -- these matter most for SEO.

Line 1-2: What the video is / what set / what you are doing
Line 3-5: Context -- sourcing, market right now, why watch
Line 6+: Links
- Shop: hokocollectables.com
- eBay: [owner to add eBay store URL]
- Instagram: @hokocollectables
- TikTok: @hokocollectables
- Sell your collection / buyback: hokocollectables.com or DM
- Use code HOKO10 for 10% off your first Shopify order
Timestamps: add if video is over 3 minutes (owner fills in)

### 3. Tags (15-20)

Mix of:
- Specific: [full set name], [set name + booster box], [set name + Australia]
- Broad: pokemon tcg, pokemon cards australia, tcg opening, pokemon booster box
- Long-tail: [set name + worth it], [set name + price australia], [set name + pull rates]
- Brand: hoko collectables, hokocollectables

### 4. Thumbnail brief

- Main subject: box / hit card / reaction shot
- Text overlay: set name + HOKO Collectables (always include the brand)
- Colours: bold, high contrast, dark background preferred
- Avoid: cluttered layouts, tiny text, faces cut off

---

## Audit mode (for existing videos)

For each existing video, flag as OK / Needs update / Missing entirely:
- Is title searchable? (set name + Australia + year)
- Does description have timestamps?
- Does it have shop link and HOKO10 code?
- Are tags present and relevant?
- Does thumbnail have text overlay with set name?
