# Whatnot Show Prep

Generate everything needed to go live on Whatnot.

## Inputs

- **Show date/time (AEST):**
- **Show type:** Singles / Breaks / Sealed / Mixed / Mystery
- **Featured products / sets:**
- **Starting bid baseline (AUD):**
- **Estimated duration:**

## Output

1. **Show title** — catchy, search-optimised for Whatnot AU TCG viewers, under 50 chars
2. **Show description** — 3–4 lines: what they'll see, why tune in, any giveaways
3. **Run-of-show** — time-stamped bullet plan: intro (1 min), warm-up lots, main lots, breaks, sealed reveals, sign-off
4. **Lot list** — table with: order # / item / starting bid / 1-line talk-track to say while showing it
5. **Pinned chat messages** — 3 to rotate: welcome new viewers, plug HOKO10 + Shopify, repeat shipping policy
6. **Post-show checklist** — invoices, packing slips, social recap post draft, restock note

Tone on stream: collector hosting mates, not a salesperson. Honest about condition, generous with knowledge.
