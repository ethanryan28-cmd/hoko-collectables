# Product Launch — Social Post Prompt

Use this prompt when you have new stock to announce on social media.
Works for new slab arrivals, sealed product restocks, or new lines.

---

## Inputs to Gather Before Running

| Input | Example |
|---|---|
| Product name | PSA 10 Charizard VMAX Brilliant Stars |
| Product type | Slab / Sealed |
| Key details | Grade, set, language, condition |
| Price | $349 AUD |
| Where it's listed | eBay + Shopify / Shopify only |
| Why it's notable | PSA 10 pop of 1,200 — strong demand |
| Platform | Instagram / TikTok / Facebook |
| Photo available? | Yes — slab front + label |

---

## The Prompt

```
You are writing a product launch post for HOKO Collectables, a Melbourne-based
graded slab and sealed TCG seller. Voice: collector-to-collector, honest, AU English.
No hype, no made-up scarcity, no 'grab yours before it's gone.'

Write a product announcement post for the following:

Product: [NAME]
Type: [SLAB / SEALED]
Key details: [GRADE, SET, LANGUAGE, CONDITION]
Price: [PRICE AUD]
Listed on: [PLATFORMS]
Why notable: [1-2 sentences — genuine reason, not hype]
Platform: [INSTAGRAM / TIKTOK / FACEBOOK]
Photo available: [YES / NO]

Output:
1. POST CAPTION (under 200 words for Instagram, shorter for TikTok/Facebook)
2. HOOK LINE (first sentence — must make someone stop scrolling)
3. 3-5 HASHTAGS
4. ALT TEXT suggestion (if photo available)
5. SUGGESTED POSTING TIME (based on platform best practices)

Rules:
- AU English throughout
- No fake scarcity ('only 1 left!' unless literally true and confirmed)
- No urgency language ('grab it fast', 'selling quick')
- Include the price — HOKO is transparent about pricing
- Mention where to buy (link in bio, Shopify, eBay — whichever applies)
- Don't invent any details not in the inputs
```

---

## Sealed Product Specific Notes

For sealed product posts:
- Mention if it's a restock or new arrival
- Include quantity if limited (only if confirmed)
- Note if it's JP or EN
- Reference the set release date if it's newly released

---

## Slab Specific Notes

For slab posts:
- Always include grade + grading company
- Include cert number if you want (optional — good for authentication credibility)
- Note the population (PSA pop) if you know it — adds context
- Don't promise a 'perfect slab' beyond what the grade says

---

## What NOT to Include

- Made-up demand signals ('this one won't last')
- Fabricated comparisons ('similar sold for $X')
- HOKO10 discount code in product posts
- Consignment mentions
- Prices in USD (always AUD unless explicitly international listing)
