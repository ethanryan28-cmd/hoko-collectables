# Social Caption — Instagram / TikTok

Generate a caption (and TikTok script if applicable) for a HOKO post.

## Inputs

- **Platform:** Instagram / TikTok
- **Post type:** New arrival / Restock / Whatnot show announcement / Unboxing / Behind-the-scenes / Customer feature / Buyback callout
- **Subject:** (what's the post actually about)
- **Call to action:** link in bio / DM / Whatnot link / shop link / none

## Output

1. **Hook** — first line, must stop the scroll
2. **Body** — 2–4 short lines, plain English, max 1–2 emojis (only if they earn their place)
3. **CTA** — one line
4. **Hashtags** — 8–12 relevant ones, mix broad (#pokemontcg #onepiecetcg #mtg) with AU-specific (#pokemonaustralia #tcgaustralia #melbournecollectors)

**If TikTok:** also draft a **3-beat video script** (Hook → Show → Pitch), under 30 seconds spoken, with on-screen text suggestions.

Tone: real, collector-to-collector. No "GUYS! YOU WON'T BELIEVE!". Don't write like a marketing agency.
