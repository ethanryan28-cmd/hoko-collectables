## Instructions for Claude

Read the transcript above carefully. Then:

### Step 1 — Extract, don't invent

Pull out only what was actually said. Never fabricate card names, prices, dates, grading details, or any specifics not mentioned. If a detail is unclear or missing, leave a [FILL IN: ...] placeholder.

### Step 2 — Categorise

Sort extracted content into whichever of these buckets apply:

| Bucket | Examples |
|--------|---------|
| Listings to draft | "I've got a Charizard ex, near mint, thinking $80" |
| Social posts | "Just opened a box, pulled two hits" |
| Whatnot show ideas | "Thinking of doing a live Friday night" |
| Customer DMs / replies | "That guy asking about the Battle Partners lot" |
| Pricing / buyback decisions | "Bloke wants to sell his binder, I said no singles" |
| Restock / inventory notes | "Getting another 6 Mega Dream boxes next week" |
| To-dos / reminders | "Need to photograph the PSA slabs today" |
| Strategic thoughts | "Thinking consignment could be bigger than buybacks long-term" |

### Step 3 — Produce structured output

One section per bucket. Within each:
- **Listings:** Draft title + description using routines/prompts/ebay-listing.md or routines/prompts/shopify-sealed.md format. Add [FILL IN] where details are missing.
- **Social posts:** Draft caption + hashtags using routines/prompts/social-caption.md format. Label the content pillar (The Pull / The Drop / The Story / The Tip / The Hit-list).
- **Whatnot ideas:** Note the concept with a suggested show title and any prep needed.
- **DM replies:** Draft a reply in HOKO voice — collector-to-collector, honest, AU English.
- **Pricing/buyback:** State the decision clearly. If it involves a buyback quote, flag whether it fits the rules (PSA/BGS/CGC slabs $100+, sealed only, no singles, no HGA/GMG).
- **Restock notes:** Confirm SKU, quantity, estimated arrival. Flag if it needs a Shopify listing or SKU watchlist update.
- **To-dos:** Clean numbered list.
- **Strategic thoughts:** File as a note. No action required unless owner asks.

### Step 4 — End with a tight to-do list

Numbered list of what owner needs to review, approve, or action when back at the PC. Max 10 items.

## Rules

- AU English throughout all drafts.
- Never post anything on owner's behalf — always hand back for approval.
- If the transcript mentions a buyback enquiry: check against buyback rules (75-80% FMV, PSA/BGS/CGC only, $100+ per item, bank branch/storage facility meets, bank transfer same day, 48hr quote). Flag any part that doesn't fit.
- If the transcript mentions a consignment enquiry: note it as Phase 2 casual inbound only. Do not send formal contract yet — just draft a holding reply.
- Tone in all drafts: collector talking to another collector. Not a retail assistant.
