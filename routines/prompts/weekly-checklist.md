# Weekly Checklist

HOKO Collectables weekly operations checklist.
Run through this each week — takes about 30-45 minutes total.

Print or open in a text editor and check off items as you go.

---

## Listings

- [ ] Check all active eBay listings — are prices still accurate vs current market?
- [ ] Check all active Shopify listings — any zero-stock listings still live?
- [ ] Run `check_inventory.py` — archive zero-stock active variants
- [ ] Review new arrivals — are photos and descriptions done?
- [ ] Any listings to relist (expired or ended early)?

---

## Pricing

- [ ] Check 3-5 slabs against recent eBay AU sold prices
- [ ] Check 2-3 sealed products against current market
- [ ] If pricing is off by 10%+ either way — update now
- [ ] Check `price_check.py` output if you ran it this week

---

## Orders

- [ ] All orders from the past 7 days dispatched?
- [ ] Any tracking not scanned yet — follow up with Australia Post
- [ ] Any unresolved eBay cases or buyer messages?
- [ ] Any refund requests — assess and respond
- [ ] Feedback received this week — respond to any negative feedback

---

## Buyback

- [ ] Any open buyback enquiries — have you responded within 48 hours?
- [ ] Any outstanding valuations to complete?
- [ ] Any agreed deals to follow up on?

---

## Content

- [ ] Post at least 1 TikTok or Reel this week (use `tiktok-content.md` if needed)
- [ ] Any new arrivals worth a quick photo post?
- [ ] Check Instagram/TikTok comments — any questions to reply to?
- [ ] Whatnot show coming up? — check `whatnot-setup-guide.md` if within 7 days

---

## Admin

- [ ] Log any expenses from the past 7 days (Xero or spreadsheet)
- [ ] Any invoices or supplier payments due?
- [ ] Stock on hand — anything running low that needs reordering?
- [ ] Any grading submissions to check on (PSA/BGS/CGC order tracking)

---

## End of Week Check

- [ ] How many orders this week vs last week?
- [ ] Any issues or patterns worth noting?
- [ ] Anything to prep for next week?

---

## Monthly Triggers

These only apply in the last week of the month:

- [ ] Run `end-of-month-review.md` prompt
- [ ] BAS due? — check `tax-and-gst-guide.md` for dates
- [ ] Draft monthly newsletter — use `email-newsletter.md` prompt
- [ ] Review eBay seller health dashboard

---

## Notes

Keep this checklist quick — if an item takes more than 5 minutes, schedule it separately.
The goal is to catch issues early, not to resolve everything in one sitting.
