# Dispatch Guide

HOKO Collectables packing and dispatch standards.
Follow this every time an order is sent — consistency protects the business.

---

## Before You Pack

1. Confirm the order details — buyer name, address, item, platform
2. Photograph the item before packing (front and back)
3. Check the slab or sealed product for any new damage
4. Confirm the shipping method selected by the buyer

---

## Packing Standards by Product Type

### PSA / BGS / CGC Slabs

| Slab Value | Packaging | Notes |
|---|---|---|
| Under $200 | Bubble wrap + rigid slab sleeve + padded envelope | Minimum standard |
| $200–$500 | Bubble wrap + rigid toploading case + small box | Double wrap the slab |
| $500+ | Bubble wrap + rigid toploading case + small box + fragile sticker | Consider signature on delivery |
| $1,000+ | All of above + tracked Express + signature required | Never standard post |

**Always:**
- Use a rigid toploading case or hard case — never just bubble wrap alone
- The slab must not be able to move inside the package
- Do not use oversized boxes that allow internal movement

---

### Sealed TCG Product

| Product | Packaging | Notes |
|---|---|---|
| Single ETB / booster box | Bubble wrap + snug cardboard box | Protect corners |
| Multiple ETBs | Individual bubble wrap + outer box with packing fill | |
| Booster packs (loose) | Sealed in bag + bubble wrap + padded envelope | |

**Always:**
- Protect all 4 corners — most damage happens to corners in transit
- Sealed product should not be able to slide inside the package
- Add a 'Do Not Bend' or 'Fragile' sticker on sealed boxes

---

## Dispatch Checklist

Run through this before sealing the package:

- [ ] Item photographed (both sides)
- [ ] Item matches the order (correct card, correct grade, correct variant)
- [ ] Packaging meets the standard for this item's value
- [ ] Package insert included (optional — use `routines/prompts/package-insert.md`)
- [ ] Address label is correct and legible
- [ ] Shipping method matches what buyer paid for
- [ ] Tracking number copied — ready to add to order

---

## Shipping Providers

| Method | Use Case | Tracking | Signature |
|---|---|---|---|
| Australia Post Regular | Under $100 | Yes | No |
| Australia Post Express | $100–$500 | Yes | Optional |
| Australia Post Express + Signature | $500+ | Yes | Yes |
| StarTrack / Courier | Bulky sealed orders | Yes | Yes |

---

## After Dispatch

1. Add tracking number to the order (eBay / Shopify)
2. Mark order as dispatched on the platform
3. Keep the tracking receipt until the buyer confirms delivery
4. Keep pre-pack photos for 60 days minimum

---

## High-Value Orders ($500+)

For orders over $500:
- Always use Express + signature on delivery
- Consider adding transit insurance (see `docs/insurance-guide.md`)
- Take a video of the packing process if practical
- Never leave at door — signature required

---

## International Orders

- Declare accurate value on customs form — never undervalue
- Use Australia Post International Express for high-value items
- Clearly state 'Collectible Trading Card — Graded' on customs form
- Buyer is responsible for import duties — note this in your listing
