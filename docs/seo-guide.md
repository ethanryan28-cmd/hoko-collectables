## 2. Shopify Product Title Formulas

Title is the #1 on-page SEO signal. Follow these formulas exactly — don't deviate for aesthetic reasons.

### Sealed TCG Product

```
[Product Name] | [Game] TCG | [Region] Sealed | HOKO Collectables
```

**Examples:**
- `Pokémon Scarlet & Violet 151 Booster Box | Pokémon TCG | English Sealed | HOKO Collectables`
- `Charizard ex Super Premium Collection | Pokémon TCG | English Sealed | HOKO Collectables`
- `One Piece Romance Dawn Booster Box | One Piece TCG | English Sealed | HOKO Collectables`
- `Pokémon Eevee Heroes Booster Box | Pokémon TCG | Japanese Sealed | HOKO Collectables`

### Graded Slab

```
[Card Name] [Set/Year] [PSA/BGS/CGC] [Grade] | [Game] | [Holo/1st Ed if applicable] | HOKO Collectables
```

**Examples:**
- `Charizard Base Set 1999 PSA 9 | Pokémon TCG | Shadowless Holo | HOKO Collectables`
- `Black Lotus Alpha PSA 7 | Magic: The Gathering | HOKO Collectables`
- `Mew ex 151 PSA 10 | Pokémon TCG | HOKO Collectables`

## 3. Shopify Meta Description Templates

Keep between 140–160 characters. Include the primary keyword and a differentiator.

### Sealed product:

```
[Product name] — factory-sealed, sourced through our regional supplier network. Free AU shipping over $100. Collector-run Melbourne shop.
```

**Example:**


**Example:**

## 5. eBay Title Formula (SEO-optimised)

eBay search is keyword-matching, not semantic — every word counts. Use all 80 characters.

```
[Card/Product Name] [Set] [Year if vintage] [Grade/Condition] [PSA/BGS/CGC] [Grade #] [Region] [Sealed/1st Ed/Holo]
```

**Sealed example (80 chars):**
> `Pokemon 151 Booster Box Scarlet Violet English Sealed TCG Factory Sealed NEW AU`

**Graded slab example (80 chars):**
> `Charizard Base Set 1999 PSA 9 Shadowless Holo Rare Pokemon TCG Card Graded`

**Rules:**
- Always include the game name (Pokemon/Magic/One Piece)
- Always include region (English/Japanese) for sealed
- Always include grader (PSA/BGS/CGC) and grade number for slabs
- "AU" at the end helps filter Australian search results
- Never keyword-stuff misleading terms ("rare", "ultra rare" for sealed boxes)

## 7. On-Page SEO Checklist (Per Shopify Product)

Run through this before publishing any new product:

- [ ] Title includes primary keyword (game name, product type, region)
- [ ] Meta description is 140–160 chars, includes keyword + differentiator
- [ ] Product description includes keyword in first sentence
- [ ] Alt text on all images: `[Product name] [game] [sealed/graded] [grade if applicable]`
- [ ] URL handle auto-generated from title (check: no generic numbers, includes game name)
- [ ] Tagged with correct Shopify collection (Pokémon Sealed / Graded Slabs / MTG Sealed etc.)
- [ ] Sourcing note included: "sourced through our regional supplier network" or "imported via our wholesale partners"
- [ ] Shipping + condition noted in description body

## 9. Quick Wins (Do These First)

1. **Google Search Console** — add hokocollectables.com and verify via Shopify. Free. Shows exactly which keywords you rank for today.
2. **Shopify SEO audit** — check every collection page has introductory copy (most Shopify stores skip this).
3. **Alt text on all product images** — most stores have empty alt text. Easy win.
4. **Descriptive URL handles** — e.g. `/products/pokemon-151-booster-box-english-sealed` not `/products/12345`
5. **Internal links** — from product descriptions, link to the relevant collection page ("Browse all Pokémon sealed →").
