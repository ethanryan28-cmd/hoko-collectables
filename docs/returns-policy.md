# Returns Policy

HOKO Collectables returns and refunds reference for Ethan.
This outlines HOKO's policy and maps it to eBay and Shopify obligations.

---

## HOKO Policy Overview

HOKO sells graded slabs (PSA/BGS/CGC) and sealed TCG product.
Because all items are either third-party graded or factory sealed,
returns are handled on a case-by-case basis with the following framework:

| Situation | HOKO Position |
|---|---|
| Item arrived not as described (INAD) | Full refund, buyer keeps item or return at HOKO's cost |
| Item not received (INR) | Investigate tracking; full refund if confirmed lost |
| Buyer changed mind (sealed) | No return — sealed product cannot be verified once buyer has it |
| Buyer changed mind (slab) | No return — graded cards are sold with cert number verification |
| Damaged in transit | Refund if inadequate packaging; lodge claim if carrier fault |
| Grading dispute (e.g. buyer disagrees with grade) | Not a valid return — grade is set by the grading company, not HOKO |

## eBay Returns

### eBay Policy Requirements

HOKO must follow eBay's Money Back Guarantee regardless of internal policy.
eBay MBG covers:
- Item not received
- Item not as described

eBay does NOT require acceptance of change-of-mind returns if the listing says 'No returns'.

### Recommended eBay Returns Setting

- **Returns accepted:** No (for change-of-mind)
- **eBay Money Back Guarantee:** Always applies — cannot be opted out
- **Handling time:** 1 business day

### eBay Return Process

1. Buyer opens return case in eBay Resolution Centre
2. HOKO has 3 business days to respond
3. If INAD: accept return or offer partial refund
4. If INR: check tracking, then refund if unresolved after 3 business days
5. If change of mind: decline (if returns are set to 'No')
6. If escalated to eBay: provide tracking + photos + listing evidence

See `routines/prompts/dispute-resolution.md` for response templates.

## Shopify Returns

### ACL Obligations (Australia)

Under Australian Consumer Law, HOKO must provide a remedy if:
- The item has a major fault
- The item is not as described
- The item is not fit for purpose

HOKO does NOT have to accept returns for change of mind.

### Shopify Return Process

1. Buyer emails or messages HOKO
2. Assess against ACL and HOKO policy
3. If valid: issue refund via Shopify admin (Refunds section of the order)
4. If invalid: explain clearly in writing — keep the message
5. If disputed: follow up professionally; avoid escalation

## Damaged in Transit

| Packaging | HOKO's Liability |
|---|---|
| Slab in bubble wrap only | HOKO liable — inadequate packaging |
| Slab in rigid toploading case + bubble + box | Carrier liable — lodge claim |
| Sealed product in bubble wrap only | HOKO liable |
| Sealed product in padded box | Carrier liable — lodge claim |

Always photograph items before packing. Keep photos for 60 days after dispatch.

---

## What to Avoid

- Arguing grades with buyers — the grader sets the grade, not HOKO
- Going quiet on a case — always respond within 2 business days
- Accepting returns on sealed product where tamper is possible
- Refunding without receiving the item back (unless INAD where buyer keeps it)
- Negative feedback retaliation — never respond to negative feedback with negative feedback

---

## Record Keeping

For every return or dispute:
- Screenshot the original listing
- Screenshot all buyer messages
- Note the outcome and date
- If a refund was issued, note the amount and reason

Keep records for at least 2 years (relevant for GST and ACL purposes).
